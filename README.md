[Basit ATM Simülasyonu]

Bu proje, Python ile geliştirilmiş hafif bir ATM Simülasyonu uygulamasıdır. Döngüler, koşullu mantık, hata yönetimi ve terminal arayüzü yönetimi gibi temel programlama kavramlarını uygulamalı olarak göstermektedir.

[Özellikler]

Güvenli Kimlik Doğrulama: Maksimum 3 deneme hakkı sunan kullanıcı giriş sistemi.

Bakiye Sorgulama: Mevcut hesap bakiyesini anlık olarak kontrol etme.

Hesap Detayları: Hesap sahibi bilgilerini ve örnek IBAN detaylarını görüntüleme.

Para Yatırma ve Çekme: Yetersiz bakiye kontrolü ve girdi doğrulaması ile dinamik bakiye güncellemeleri.

Temiz Kullanıcı Arayüzü: os.system kullanarak daha iyi bir kullanıcı deneyimi için otomatik terminal temizleme.

[Kullanılan Teknolojiler]

Dil: Python 3.13.5

Modüller: os (terminal temizleme), time (simülasyon gecikmeleri).

[Simple ATM Simulation]

This is a lightweight ATM Simulation project built with Python. It demonstrates core programming concepts such as loops, conditional logic, exception handling, and terminal interface management.

[Features]

Secure Authentication: User login system with a maximum of 3 attempts.

Balance Inquiry: Check your current account balance instantly.

Account Details: View owner information and mock IBAN details.

Deposit & Withdraw: Dynamic balance updates with input validation to prevent overdrafts.

Clean UI: Automatic terminal clearing for a better user experience using os.system.

[Built With]

Language: Python 3.13.5

Modules: os (terminal clearing), time (simulation delays).
